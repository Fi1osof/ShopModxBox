shopModxBox - это готовая сборка магазина для MODX Revolution
на базе следующих модулей:
-   shopModx        - основной модуль для разработки магазинов.
-   phpTemplates    - модуль, позволяющий выполнять статические MODX шаблоны как 
    обычные php-файлы.
-   modxSmarty      - Smarty-шаблонизатор для MODX-а.
-   Модуль для выгрузки каталога в Яндекс.Маркет

Это уже готовый типовой магазин, остается только изменить шаблон.

Все вопросы задавайте на http://modxclub.ru
